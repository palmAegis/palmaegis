üìä About the Dataset

This detection system uses the Oil Palm Ganoderma Detection dataset, which contains:

  • Training Set: 990 images with annotations
  • Validation Set: 96 images with annotations
  • Test Set: 48 images with annotations
  • Categories: Oil Palm Tree, Ganoderma, Ganoderma Fungus
  • Format: COCO format annotations for object detection
  • Source: Roboflow Universe (CC BY 4.0 License)

The model has been trained on this dataset to identify oil palm tree diseases, with a primary focus on Ganoderma disease detection. The dataset is located in the dataset/ folder and contains train, valid, and test subsets.
